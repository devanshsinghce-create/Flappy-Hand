# ✋ Flappy Hand – AI Edition

An interactive browser-based game inspired by Flappy Bird, controlled using real-time hand gestures via AI. Built using HTML5 Canvas and MediaPipe, this project replaces traditional controls with webcam-based hand tracking for a fun and modern gaming experience.

---

## 🚀 Features

- ✋ Hand Gesture Controls (via MediaPipe)
- 🎮 Smooth game loop & physics system
- 🧠 Real-time AI hand tracking integration
- 🎨 Rich visual effects (clouds, particles, HUD)
- 🪙 Coins & Power-ups system
- 🛡 Shield ability with timer
- 🔥 Combo scoring system
- ❤️ Lives system
- 📈 Level progression & difficulty scaling
- 💾 Local storage for high score & stats
- 🔊 Lightweight audio engine

---

## 🛠️ Tech Stack

- HTML5 Canvas
- JavaScript (Vanilla JS)
- MediaPipe Hands API
- Web Audio API
- LocalStorage API

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/flappy-hand-ai.git

2. Open the project folder

3. Run the game:
   - Open "Mini game.html" in your browser
   - Allow camera access when prompted

4. Raise your hand above your head to start playing

---

## 🎮 Gameplay Instructions

- Move your hand up/down to control the bird
- Avoid hitting pipes
- Collect coins for extra score
- Grab shields to block damage
- Maintain streaks for combo bonuses
- Survive waves and increase your level
